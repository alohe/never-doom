# NeverDoom

**Your emergency calm button.** A wall of motivational quotes to ground you when life hits hard.

**[Open NeverDoom](https://alohe.github.io/neverdoom)**

## What is this?

NeverDoom is a simple, open-source static site that shows a grid of motivational quote images. Stoic philosophy, anime wisdom, raw mindset energy -- whatever gets you through the moment.

No tracking. No cookies. No sign-ups. Just open the page and breathe.

## Contributing

We'd love more quotes. See [CONTRIBUTING.md](CONTRIBUTING.md) for how to add yours.

The short version:

1. Fork this repo
2. Add your image to the `images/` folder
3. Add an entry to `quotes.json`
4. Open a pull request

## Running locally

Just open `index.html` in your browser. No build step, no dependencies.

Or serve it locally:

```bash
npx serve .
```
